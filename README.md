# Инструкция по использованию
## 1. Скопируйте репозиторий
```
git clone https://github.com/Larionov-VA/CompMath_4342.git
```
## 2. Создайте свою ветку
```
git checkout -b <Фамилия>_<Имя>
```
## 3. Создайте папку, далее вы будете работать только внутри нее
```
mkdir <Фамилия>_<Имя>
```
## 4. В своей папке создавайте папки под каждую отдельную ЛР
```
cd <Фамилия>_<Имя>
```
```
mkdir ЛР_<Номер работы>
```
